Formatação completa
via pendrive bootável

Guia passo a passo para criar um pendrive com imagem ISO e realizar a formatação e reinstalação do Windows. Siga as fases em ordem.

⚠ Antes de começar: Confirme com o cliente que todos os arquivos importantes foram copiados para um HD externo, nuvem ou outro dispositivo. A formatação apaga tudo na partição do sistema. Não há como recuperar depois.
O que você vai precisar
💾
Pendrive de pelo menos 8 GB — de preferência 16 GB ou mais. Será formatado durante o processo.
🌐
Acesso à internet para baixar a ISO do Windows pelo site oficial da Microsoft.
🖥️
Um segundo computador funcionando para criar o pendrive bootável (ou o próprio PC, se ainda ligar).
🔑
Chave de produto do Windows (opcional para versões vinculadas à conta Microsoft ou placa-mãe).
⏱️
Tempo estimado: 40 min a 1h30, dependendo da velocidade do PC e do pendrive.
Ferramentas recomendadas
⚡
Rufus
Gravador de ISO mais utilizado. Leve, rápido e com suporte a UEFI/GPT.
Gratuito Windows
🪟
Media Creation Tool
Ferramenta oficial da Microsoft. Baixa e cria o pendrive automaticamente.
Gratuito Windows
🐧
Ventoy
Permite múltiplas ISOs no mesmo pendrive. Ideal para técnicos.
Gratuito
🔥
balenaEtcher

Fases do processo
01
Baixar a imagem ISO do Windows
Obtenha a ISO original do site da Microsoft
▼
⛔ Nunca use ISOs de sites não oficiais. ISOs modificadas podem conter malware, drivers comprometidos ou ativadores ilegais. Use sempre o site oficial: microsoft.com/pt-br/software-download/windows11
1
Acesse o site oficial Vá até microsoft.com/pt-br/software-download e escolha a versão do Windows (10 ou 11) compatível com o hardware do cliente.
2
Escolha entre ISO ou Media Creation Tool Para uso com Rufus, baixe a ISO diretamente. Para criação automática, baixe o Media Creation Tool — ele faz o download e cria o pendrive em um passo só.
3
Selecione arquitetura e idioma Escolha 64-bit (x64) para qualquer computador com mais de 4 anos. A maioria dos equipamentos atuais usa 64-bit.
4
Verifique o arquivo baixado Confirme que a extensão é .iso e o tamanho bate com o esperado (Windows 11: ~5,5 GB / Windows 10: ~4,5 GB).
02
Criar o pendrive bootável com Rufus
Gravar a ISO no pendrive de forma correta
▼
⛔ O pendrive será apagado. Certifique-se de que não há arquivos importantes nele antes de continuar.
1
Abra o Rufus como administrador Clique com o botão direito no Rufus e selecione "Executar como administrador" para garantir acesso ao dispositivo.
2
Selecione o dispositivo (pendrive) No campo Dispositivo, escolha seu pendrive. Verifique a letra e o tamanho para não confundir com outro HD.
3
Selecione a ISO Clique em "SELECIONAR" e localize o arquivo .iso baixado. O Rufus preencherá automaticamente as configurações recomendadas.
4
Configure o esquema de partição Para PCs modernos com UEFI: escolha GPT. Para PCs antigos com BIOS legada: escolha MBR. Em caso de dúvida, verifique o tipo de firmware na BIOS do equipamento-alvo.
⚙ UEFI → GPT
5
Clique em "INICIAR" e aguarde O processo leva de 5 a 15 minutos dependendo da velocidade do pendrive. Não remova durante a gravação. Ao final, o Rufus exibirá "PRONTO" em verde.
03
Configurar o boot pela BIOS
Fazer o computador iniciar pelo pendrive
▼
1
Conecte o pendrive e ligue o PC Insira o pendrive em uma porta USB direta (evite hubs). Ligue ou reinicie o computador.
2
Entre no menu de boot (F12 na maioria) Pressione F12 repetidamente logo após ligar. Se não funcionar, consulte o Guia de Teclas BIOS para a marca do equipamento.
3
Selecione o pendrive na lista de boot Procure o nome do pendrive na lista. Pode aparecer como "USB Drive", "Generic Flash Disk" ou o nome da marca do pen. Priorize a versão UEFI quando houver duas opções.
4
Desative o Secure Boot se necessário Alguns firmwares bloqueiam ISOs externas. Acesse a BIOS (não o menu de boot) → Aba Security → desative Secure Boot. Reative ao final da instalação.
04
Formatação e instalação do Windows
O ponto sem retorno — todos os dados serão apagados
▼
⛔ Última confirmação. A partir deste ponto, o conteúdo do disco será apagado. Confirme com o cliente que o backup foi realizado.
1
Aguarde o carregamento do instalador Após o boot pelo pendrive, surgirá a tela azul do setup do Windows. Selecione idioma, formato de hora e teclado. Clique em "Instalar agora".
2
Chave do produto Se o Windows estava ativado digitalmente (vinculado ao hardware), clique em "Não tenho uma chave de produto" — ele será reativado automaticamente ao conectar à internet.
3
Escolha o tipo de instalação Selecione "Personalizada: instalar somente o Windows" para formatação completa. Nunca use "Atualização" neste contexto.
4
Selecione e formate a partição do sistema Clique na partição onde o Windows estava instalado (geralmente a maior, marcada como "Primária"). Clique em "Formatar". Para limpar completamente: delete todas as partições existentes e clique em "Novo" para recriar. O instalador criará as partições necessárias automaticamente.
Excluir → Novo → Aplicar → Avançar
5
Aguarde a instalação O Windows copiará os arquivos e reiniciará 2–3 vezes. Não remova o pendrive durante esse processo. Tempo estimado: 15–30 minutos.
05
Configuração inicial e pós-instalação
OOBE, drivers e ativação
▼
1
Configure a OOBE (tela de boas-vindas) Escolha região, idioma, rede. Para contas locais no Windows 11, clique em "Opções de entrada" → "Conta offline" para evitar vinculação obrigatória à conta Microsoft.
2
Instale os drivers Acesse o Gerenciador de Dispositivos (Win + X) e verifique se há dispositivos com ponto de exclamação amarelo. Baixe os drivers do site do fabricante do notebook/placa-mãe — nunca use sites de drivers de terceiros.
3
Execute o Windows Update Vá em Configurações → Windows Update → "Verificar atualizações". Instale tudo e reinicie quando solicitado. Repita até não haver atualizações pendentes.
4
Verifique a ativação Configurações → Sistema → Ativação. Deve exibir "Windows está ativado com uma licença digital". Se não estiver, use a chave do produto ou oriente o cliente.
5
Reative o Secure Boot se foi desativado Volte à BIOS e reative o Secure Boot. Isso protege o sistema contra bootkits e malwares de nível de firmware.
Dicas para o atendimento
Boas práticas
  • Sempre fotografe a tela da BIOS antes de qualquer alteração — inclusive a ordem de boot e o Secure Boot.
  • Se o cliente tiver Windows 10 ativado, a licença digital fica vinculada ao hardware e não precisa de chave — o Windows 11 também herda essa ativação.
  • Pendrive USB 3.0 reduz o tempo de instalação à metade. Observe a cor da porta (azul = USB 3.0).
  • Ao deletar partições e recriar, o instalador cria automaticamente as partições EFI, MSR e de recuperação — não se preocupe com elas.
  • Se o PC não reconhecer o pendrive no menu de boot, tente outra porta USB ou verifique se o Fast Boot está desativado na BIOS.
  • O Ventoy é ideal para técnicos que precisam ter múltiplas versões de SO no mesmo pendrive, sem reformatar a cada ISO.
