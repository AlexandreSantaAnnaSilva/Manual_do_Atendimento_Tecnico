# 📖 Guia BIOS - Teclas de Acesso

Este projeto é uma página web simples e funcional que serve como um guia de referência rápida para teclas de acesso de BIOS de diferentes fabricantes. 

## 🚀 Demonstração
Você pode acessar a página online aqui: [CLIQUE AQUI PARA VER O SITE](https://github.io)
*(Dica: Substitua o link acima pelo link que o GitHub Pages gerou)*

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação do conteúdo.
* **CSS3**: Estilização (com foco em Dark Mode e variáveis de cores).
* **Google Fonts**: Fontes 'Space Mono' e 'Sans Serif' para melhor legibilidade.

## 💻 Como rodar o projeto localmente
1. Clone este repositório:
   ```bash
   git clone https://github.com
   ```
2. Abra o arquivo `index.html` no seu navegador.

## 📝 Funcionalidades
- [x] Listagem de teclas de acesso por marca.
- [x] Design responsivo (funciona no celular e PC).
- [x] Tema escuro (Dark Mode) nativo.
